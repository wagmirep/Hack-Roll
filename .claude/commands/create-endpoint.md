# Create API Endpoint

Create a new API endpoint with the following specifications:

**Endpoint:** $ARGUMENTS

## Requirements

1. Include proper error handling with try/catch
2. Add TypeScript interfaces for request/response
3. Include input validation
4. Add appropriate HTTP status codes
5. Follow existing patterns in the codebase

## Backend (Django)

- Create view function in appropriate app
- Add URL pattern
- Include docstring with endpoint description

## Frontend (if needed)

- Create API client function in `frontend/src/api/`
- Include TypeScript types
- Handle loading and error states

After creating, verify with `bun run typecheck`.
